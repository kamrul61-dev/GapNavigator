# Vetted Tech Learning Resources

## Python Programming
- **Title**: Official Python Tutorial
  **URL**: https://docs.python.org/3/tutorial/
  **Type**: Documentation
  **Description**: The official tutorial provided by the Python Software Foundation. Excellent for learning core syntax, data structures, control flow, and modules.
  **Skills**: Python, Coding Basics, Data Structures

- **Title**: Python for Everybody (PY4E)
  **URL**: https://www.py4e.com/
  **Type**: Course
  **Description**: A free, comprehensive course by Dr. Charles Severance. It covers basic programming logic, scraping, databases (SQL), and data visualization in Python.
  **Skills**: Python, SQL, Web Scraping, Databases

## Web Development (Frontend & React)
- **Title**: MDN Web Docs - HTML, CSS & JavaScript
  **URL**: https://developer.mozilla.org/en-US/docs/Learn
  **Type**: Documentation
  **Description**: The definitive documentation and tutorial guide for all things HTML, CSS, and modern JavaScript. Crucial for understanding browser behaviors and basic frontend.
  **Skills**: HTML, CSS, JavaScript, Frontend

- **Title**: React Official Documentation
  **URL**: https://react.dev/learn
  **Type**: Documentation
  **Description**: The newly updated React docs. It offers a hands-on tutorial and deep explanations of component-based state, hooks, and clean UI rendering practices.
  **Skills**: React, JavaScript, Frontend, Component Design

- **Title**: Full Stack Open
  **URL**: https://fullstackopen.com/en/
  **Type**: Course
  **Description**: A free, deep-dive modern web development course by the University of Helsinki. Covers React, Redux, Node.js, MongoDB, GraphQL, and TypeScript.
  **Skills**: React, Node.js, JavaScript, MongoDB, TypeScript, REST API

## SQL & Databases
- **Title**: SQLZoo Interactive Tutorial
  **URL**: https://sqlzoo.net/
  **Type**: Tutorial
  **Description**: A step-by-step interactive SQL tutorial with live queries. Highly recommended for practicing SELECT statements, JOINs, aggregations, and subqueries.
  **Skills**: SQL, Databases, Relational Algebra

- **Title**: Mode Analytics SQL Tutorial
  **URL**: https://mode.com/sql-tutorial/
  **Type**: Course
  **Description**: A well-structured SQL course covering basic, intermediate, and advanced SQL concepts (like window functions and CTEs) with a focus on data analysis.
  **Skills**: SQL, Data Analysis, Window Functions, CTEs

## Version Control & Git
- **Title**: Git Book (Pro Git)
  **URL**: https://git-scm.com/book/en/v2
  **Type**: Documentation
  **Description**: The official Pro Git book covering basic cloning, branching, merging, and advanced workflows.
  **Skills**: Git, Version Control, Collaboration

- **Title**: GitHub Skills Tutorials
  **URL**: https://skills.github.com/
  **Type**: Interactive
  **Description**: Interactive GitHub-hosted courses for learning Git commands, branching, resolving conflicts, and managing pull requests.
  **Skills**: Git, GitHub, Version Control

## Data Science & Analytics
- **Title**: Kaggle Learn Courses
  **URL**: https://www.kaggle.com/learn
  **Type**: Course
  **Description**: Free, bite-sized interactive courses covering Pandas, Data Visualization, Intro to Machine Learning, and SQL.
  **Skills**: Pandas, NumPy, Machine Learning, Data Visualization, Python

- **Title**: Pandas Official User Guide
  **URL**: https://pandas.pydata.org/docs/user_guide/index.html
  **Type**: Documentation
  **Description**: Extensive official documentation for the Pandas library in Python, explaining DataFrames, Series, grouping, and indexing.
  **Skills**: Pandas, Python, Data Cleaning

## Project Management & Agile
- **Title**: Agile Alliance Resources
  **URL**: https://www.agilealliance.org/agile101/
  **Type**: Documentation
  **Description**: The central hub for Agile methodology. Explains Scrum, Kanban, User Stories, and Agile practices.
  **Skills**: Agile, Scrum, Kanban, Project Management

- **Title**: Google Project Management Certificate
  **URL**: https://www.coursera.org/professional-certificates/google-project-management
  **Type**: Course
  **Description**: Google's official introductory training course for professional project management. Covers Scrum, Agile, and budget planning.
  **Skills**: Project Management, Agile, Scrum
